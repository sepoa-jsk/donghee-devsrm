## 1. 배경 및 목적

### 1.1 배경
- 자동차 산업의 모델 다양화, 전동화 가속으로 신규 개발 부품 수가 증가하고 개발 리드타임이 단축되고 있다.
- 동희산업은 다수의 차종 프로젝트를 동시 진행하며, 개발구매 단계의 원가/품질/일정 통제가 핵심 경쟁력이다.
- 기존 양산 ERP의 구매단가/거래처 관리만으로는 **개발 단계의 목표가 설정, LME 연동 재료비 산출, 견적 Breakdown 비교, 다부서 검토** 통제가 어렵다.

### 1.2 목적
고객 RFQ 접수 단계부터 ERP 양산 전환까지의 개발구매 전 과정을 **품번 기준**으로 연결하여 다음을 달성한다.

1. 협력사 견적 전 **내부 목표 구매단가**를 선행 설정
2. **LME 기반 재료비 자동 산출**로 견적 타당성 검증 기반 확보
3. 협력사 견적의 **Breakdown 비교**로 협상 근거 확보
4. **설계·품질·영업** 검토를 결재 흐름에 통합하여 가격 외 리스크 통제
5. 도면/LME/환율 **변경 추적**으로 단가 변경 근거 보존
6. ERP 양산 전환 시 **체크리스트 통제**로 데이터 무결성 확보

### 1.3 비범위 (Out of Scope)
- 양산 단계의 발주/입고/검수 (기존 ERP 기능)
- 회계/세무 처리
- 협력사 등록·평가의 전체 라이프사이클 (본 시스템은 견적 단계 평가만)

## 3. 기능 요구사항 (FR)

### FR-01 고객 RFQ 등록 (영업)
- FR-01-1 고객 RFQ 번호, 차종, 요청 품목, 고객 목표 판매가, 연간수량, SOP 입력
- FR-01-2 사양서/도면/RFQ 문서 첨부
- FR-01-3 등록 시 개발·설계·구매·품질에 알림 발송
- FR-01-4 상태: RFQ등록 → 프로젝트전환대기 → 검토요청

### FR-02 개발 프로젝트 생성 (개발 PM)
- FR-02-1 고객 RFQ에서 "프로젝트 생성" 시 기본정보 자동 복사
- FR-02-2 부서별 담당자 지정 (개발/설계/구매/품질/영업)
- FR-02-3 주요 마일스톤 일정 등록 (도면등록 ~ SOP)
- FR-02-4 종합 대시보드: 일정/원가/구매/품질/설계/영업 현황 표시

### FR-03 품번·BOM·도면 관리 (설계)
- FR-03-1 품번/품명/도면번호/리비전/재질/중량(순제품/투입/스크랩) 등록
- FR-03-2 BOM 트리 구성, 각 구성품의 구매대상 여부 지정
- FR-03-3 도면 파일 첨부 및 리비전 이력 관리
- FR-03-4 LME 연동 소재(Aluminum/Copper/Steel 등) 매핑

### FR-04 목표가 설정 (구매·영업·개발)
- FR-04-1 입력: 고객 목표 판매가, 목표 마진율, 내부비용
- FR-04-2 자동 계산: 허용 총원가, 목표 구매단가
- FR-04-3 산정 근거 기록 (고객가 역산 / 유사품 기준 / 벤치마크)
- FR-04-4 승인 워크플로우 (임시 → 검토중 → 승인)
- FR-04-5 **목표가는 협력사 포털에 절대 노출 안 됨**

### FR-05 LME 기준정보 관리 (원가관리)
- FR-05-1 소재별 LME 기준가, 환율, 프리미엄, 부대비를 월/기간 단위로 등록
- FR-05-2 소재단가 자동 계산: `LME × 환율 ÷ 1000 + 프리미엄 + 부대비`
- FR-05-3 적용 기간 충돌 시 경고
- FR-05-4 외부 LME API 연동 (Phase 2, 일별 동기화)

### FR-06 재료비 산출 (구매·원가관리)
- FR-06-1 공식: `재료비 = 투입중량 × 소재단가 − 스크랩중량 × 스크랩회수단가`
- FR-06-2 도면 리비전, LME 기준월, 환율 정보 함께 기록 (스냅샷)
- FR-06-3 작성/검토/승인 결재

### FR-07 목표가 대비 원가 분석
- FR-07-1 예상 구매단가 = 재료비 + 가공비 + 물류비 + 관리비/이윤
- FR-07-2 목표가 vs 예상 비교, 차액 표시
- FR-07-3 상태 판정: 목표이내 / 주의 / 초과 / 재검토필요
- FR-07-4 초과 시 설계·구매·영업·개발에 검토 요청 발송, 의견 수집

### FR-08 구매대상 품목 확정
- FR-08-1 BOM 중 RFQ 대상 품목 선택
- FR-08-2 목표가 초과 품목은 RFQ 시 **Breakdown 제출 필수** 자동 설정

### FR-09 RFQ 생성·발송
- FR-09-1 대상 협력사 다중 선택, 견적 마감일 설정
- FR-09-2 RFQ 본문에 LME 기준월, 기준 LME가, 기준 환율, 기준 소재단가 자동 포함
- FR-09-3 협력사 입력 필수 항목 강제: 양산단가, 재료비, LME, 환율, 프리미엄, 투입중량, 스크랩, 가공비, 물류비, 관리비/이윤, 금형비, 납기
- FR-09-4 발송 시 협력사 포털 알림

### FR-10 협력사 견적 제출 (협력사 포털)
- FR-10-1 RFQ 본문 확인, 도면 다운로드
- FR-10-2 Breakdown 입력 폼, 항목 누락 시 제출 불가
- FR-10-3 제출 후 수정은 마감 전까지 가능, 이력 보존

### FR-11 견적 Breakdown 비교
- FR-11-1 협력사 N개 가로 비교 + 내부 산출값 컬럼
- FR-11-2 항목별 차이 시각화 (높음/낮음/확인필요)
- FR-11-3 종합 판단 영역: 가격/품질/납기 매트릭스
- FR-11-4 협력사 간 동일 도면·동일 LME 기준 검증

### FR-12 다부서 검토
- FR-12-1 설계 검토: 도면 적합성, 중량 산정 근거, 공법 적정성
- FR-12-2 품질 검토: 협력사 품질등급, IATF 16949, 불량률, PPAP 이력
- FR-12-3 영업 손익 검토: 견적단가 기준 마진 시뮬레이션

### FR-13 협상 이력 관리
- FR-13-1 최초 견적 → 협상 후 단가 비교 (항목별)
- FR-13-2 협상 근거 입력 필수
- FR-13-3 원가절감안 등록 (담당, 예상 효과, 상태)

### FR-14 업체 선정 품의
- FR-14-1 결재 문서 자동 생성 (최초가, 협상가, 목표가 대비, 선정 사유)
- FR-14-2 결재선: 구매담당 → 구매팀장 → 설계 → 품질 → 개발PM → 영업 → 최종승인
- FR-14-3 반려 시 사유 입력 필수, 직전 단계 복귀

### FR-15 샘플·금형·PPAP 관리
- FR-15-1 일정: 금형설계 → 제작 → T0 샘플 → 입고 → 검사
- FR-15-2 PPAP 문서: 검사성적서, 재질성적서, 공정흐름도, Control Plan, FMEA
- FR-15-3 문서 상태: 미제출/제출완료/반려/승인
- FR-15-4 샘플 단계 도면 변경 시 재료비 자동 재계산

### FR-16 확정단가 등록
- FR-16-1 단가 + 기준 스냅샷(LME, 환율, 프리미엄, 도면 리비전) 동시 저장
- FR-16-2 적용 시작/종료일 설정
- FR-16-3 단가 조정 조건 등록 (예: LME ±5% 이상 변동 시 재협의)

### FR-17 ERP 양산 전환
- FR-17-1 체크리스트 전 항목 충족 시에만 전송 가능
- FR-17-2 ERP 전송 데이터: 품번, 협력사, 단가, 유효기간, MOQ, 거래가능 여부
- FR-17-3 전송 결과 로그 보존

### FR-18 변경관리
- FR-18-1 설계 변경: 리비전 변경 시 재료비 자동 재산출, 영향 단가 시뮬레이션
- FR-18-2 LME 변동: 기준가/환율 변경 시 영향 품번 리스트와 증감액 자동 산출
- FR-18-3 변경 시 영향 부서 자동 알림, 재협의 트리거

## 5. 데이터 모델 개요 (Phase 2 사전 정의)

핵심 엔티티만 나열. ERD는 별도 문서로 확장.

```
Customer ─< Project ─< Part ─< Drawing(Rev)
                       │
                       ├─< TargetPrice (목표가 스냅샷)
                       ├─< MaterialCost (재료비 산출)
                       └─< RFQ ─< RFQItem ─< Quotation ─< QuoteBreakdown
                                              │
                                              └─< NegotiationLog
                                              └─< Selection (결재)
                                              └─< FinalPrice (확정단가)
                                              └─< PPAP
LMEReference (소재·월별)
FxRate (환율)
Supplier ─< SupplierQuality
```

## 9. 검증 시나리오 (데모 기준)

데모는 다음 12개 흐름을 H1110-T100 단일 품번으로 시연한다:

1. 고객 RFQ 등록 → 프로젝트 자동 생성
2. 설계 BOM/도면 등록, 중량 입력
3. 목표가 설정 → 12,100원 산출
4. LME 등록 (Al 2,400 USD, 환율 1,350) → 소재단가 3,640원/kg
5. 재료비 산출 → 11,168원
6. 목표가 대비 분석 → **+2,368원 초과** 경고
7. RFQ 발송 (A/B/C사)
8. 협력사 견적 접수 (A사 Breakdown)
9. 견적 비교 → A사 LME·프리미엄·투입중량 차이 시각화
10. 설계·품질·영업 검토 → 손익 -1,628원 경고
11. 협상 → 14,500원으로 인하, 이력 기록
12. 업체 선정 결재 → 확정단가 등록 → ERP 전환 체크리스트
13. (추가) LME Al 2,700 상승 시뮬레이션 → 재료비 +1,382원 자동 반영
