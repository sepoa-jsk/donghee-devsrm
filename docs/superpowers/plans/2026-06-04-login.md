# 00-01 로그인 화면 구현 계획

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** DevSRM 진입 로그인 화면(00-01)을 사이드바 없는 전체화면 중앙 카드로 구현하고 index.html / PROGRESS.md를 갱신한다.

**Architecture:** 다른 화면과 달리 `sidebar.js`를 로드하지 않는 standalone 전체화면. tokens.css의 색·폰트 변수만 재사용하고 자체 중앙 카드 레이아웃을 정의한다.

**Tech Stack:** Standalone HTML, tokens.css, lucide CDN, vanilla JS.

  function doLogin(e) {

    var pw = document.getElementById('login-pw');
    var ic = document.getElementById('pw-toggle');
    var isPw = pw.type === 'password';
    pw.type = isPw ? 'text' : 'password';
    ic.setAttribute('data-lucide', isPw ? 'eye-off' : 'eye');
