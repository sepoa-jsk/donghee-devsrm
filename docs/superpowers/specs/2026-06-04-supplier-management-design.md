## 09-01 협력사 사전 등록·평가 (상세)

**파일:** `mockups/09_supplier/09-01_supplier_register_detail.html`
**data-menu:** 09-01

### 상단 액션
- page-title-bar: "협력사 사전 등록·평가" + [목록]→09-01_supplier_register.html + [반려](btn-danger, 모달) + [승인](btn-primary, showToast)

### 섹션 1 — 업체 기본정보 (form-row, 일부 입력 가능)
- 업체코드 VND-0064 / 업체명 I사 (금강정밀) / 대표 김금강 / 사업자번호
- 소재지 / 품목군 select(알루미늄 가공) / 신청일 2026-05-20 / 담당자 이구매

### 섹션 2 — 사전 평가 체크리스트 (4개 영역)
영역별 평가 테이블(eval-table). 각 영역 3~4개 항목, 항목별 배점·점수 입력, 영역 소계 자동.

**재무 (배점 25)**
- 자본금·부채비율 / 최근 3년 매출 추이 / 신용등급
**기술 (배점 25)**
- 보유 설비 적합성 / 기술 인증(IATF 등) / R&D 역량
**품질 (배점 25)**
- 품질시스템(ISO/IATF) / 불량률 실적 / 클레임 이력
**생산 (배점 25)**
- 생산능력(CAPA) / 납기 준수율 / 공급 안정성

각 영역 소계 + 종합 점수(100점 만점) 자동 산출. 합격 기준선 70점 표시(미달 시 빨강).
I사 예시: 재무 20 / 기술 22 / 품질 21 / 생산 19 = 82점 (합격)

### 섹션 3 — 심사 결재선 (수평 4단계)
구매담당(이구매, done) → 구매팀장(이팀장, prog) → 품질담당(최품질, wait) → 구매부장(박부장, wait)

### 반려 모달
반려 사유 select(재무 부적격/기술 미달/품질시스템 미흡/기타) + textarea + [취소][반려 확정]

## 09-02 협력사 사후 평가 (Q-5스타)

**파일:** `mockups/09_supplier/09-02_supplier_eval.html`
**data-menu:** 09-02

### 상단 안내 + 범례
- page-title-bar: "협력사 사후 평가 (Q-5스타)" + [평가 저장](btn-primary, showToast)
- 평가 기준 안내 바: 5개 항목 Q(품질)/D(납기)/P(가격)/T(기술)/S(시스템), 각 1~5점
- 등급 범례: S(4.5+) / A(3.5+) / B(2.5+) / C(1.5+) / D(미만) — 각 등급 색상 점

### 분기 선택 탭
2026 1Q / 2Q / 3Q / 4Q (탭, 3Q 기본 활성). 탭 전환 시 그리드 데이터 갱신(데모는 3Q만 실제, 나머지는 동일/빈값 처리 가능).

### 평가 그리드 (협력사 × 항목)
| 협력사 | 품질(Q) | 납기(D) | 가격(P) | 기술(T) | 시스템(S) | 평균 | 등급 | 조치사항 |
|--------|---------|---------|---------|---------|-----------|------|------|----------|
점수 input(1~5, 0.1 단위 허용). 평균·등급 자동 산출. 조치사항은 등급에 따라 자동 텍스트.

**Mock 데이터 (2026 3Q)**
```
A사 (동희부품)   | 4.8 | 4.7 | 4.3 | 4.6 | 4.7 → 평균 4.62 → S등급 → 우선 발주
B사 (한국알미늄) | 4.2 | 4.0 | 4.4 | 3.8 | 4.1 → 평균 4.10 → A등급 → 우선 발주 후보
C사 (신흥정공)   | 3.5 | 3.8 | 4.0 | 3.4 | 3.6 → 평균 3.66 → A등급 → 우선 발주 후보
D사 (대원금속)   | 2.8 | 3.0 | 3.5 | 2.6 | 2.9 → 평균 2.96 → B등급 → 정상 거래
```
(검증: A사 (4.8+4.7+4.3+4.6+4.7)/5 = 23.1/5 = 4.62 → S / B사 20.5/5=4.10 → A / C사 18.3/5=3.66 → A / D사 14.8/5=2.96 → B)

등급 자동산정 함수: 평균 ≥4.5 S / ≥3.5 A / ≥2.5 B / ≥1.5 C / else D
조치사항 매핑: S→"우선 발주", A→"우선 발주 후보", B→"정상 거래", C→"경고·개선 계획", D→"거래 중단 검토"
등급 셀 색: S/A=ok(초록), B=prog(파랑), C=warn(노랑), D=bad(빨강)

### 하단 — 등급 분포 요약
S: 1개사 / A: 2개사 / B: 1개사 / C: 0 / D: 0 (현재 그리드 기준 자동 집계)
