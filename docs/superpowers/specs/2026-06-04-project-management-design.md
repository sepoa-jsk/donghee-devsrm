## 01-05 프로젝트 일정/마일스톤

**파일:** `mockups/01_project/01-05_milestone.html`
**data-menu:** 01-05

### 상단 액션
- page-title-bar: "프로젝트 일정/마일스톤" + [프로젝트 상세]→01-04_project_detail.html + [일정 저장](showToast)

### 프로젝트 컨텍스트 바 (gray)
프로젝트: PJT-520PHEV-FT-2026 / 품번: H1110-T100 / 고객: 현대 520 PHEV / PM: 김개발

### KPI 카드 4장
- 전체 진척률 45%
- 완료 마일스톤 3 / 9
- D-Day (SOP 2026-10-01 기준, 작성일 2026-06-04 → D-119)
- 지연 작업 1건 (over 카드, 빨강)

### 마일스톤 카드 스트립 (가로 9단계)
도면/BOM 등록(완료) → 목표가/재료비(완료) → RFQ 발송(완료) → 견적/업체선정(진행) → 금형/샘플(예정) → APQP(예정) → PPAP(예정) → 양산성 검증(예정) → SOP(예정)
- 각 카드: 단계명 + 기준일 + 상태 점(완료 ok / 진행 prog / 예정 wait)

### 작업 상세 + 간트 통합 테이블 (gantt-table)
좌측 고정 컬럼: 작업명 / 담당 / 시작일 / 종료일 / 진척% / 상태
우측: 월별 간트(1월~10월), g-bar 막대 done/active/plan/delay
작업 행(01-04 간트 8작업 계승 + 담당/일정/진척 추가):
```
작업명               담당   시작       종료       진척  상태   간트
도면/BOM 등록        정설계 2026-01-05 2026-02-20 100%  완료   1~2월 done
목표가/재료비 산출    한원가 2026-02-01 2026-03-10 100%  완료   2~3월 done
개발구매 RFQ 발송     김개발 2026-03-01 2026-04-15 100%  완료   3~4월 done
견적 비교/업체 선정   김개발 2026-04-01 2026-05-20 70%   지연   4~5월 delay (지연)
금형/샘플 제작        최금형 2026-05-15 2026-07-31 10%   진행   5~7월 active
APQP/품질 준비        이품질 2026-06-01 2026-08-31 0%    예정   6~8월 plan
PPAP 승인            이품질 2026-08-01 2026-09-15 0%    예정   8~9월 plan
SOP 준비             김개발 2026-09-01 2026-10-01 0%    예정   9~10월 plan
```
- 상태 점: 완료(ok)·진행(prog)·지연(bad)·예정(wait)
- g-bar: done=navy, active=blue, plan=연한 파랑 외곽선, delay=빨강. 현재월(6월) 컬럼 강조(cur).
- 진척% 셀: 가는 막대 + %

### 범례
완료(navy) / 진행(blue) / 예정(plan) / 지연(red)

간트 CSS는 01-04의 `.gantt` / `.g-bar done/active/plan` 패턴 계승 + `.g-bar.delay { background:var(--status-bad); }` 추가.
