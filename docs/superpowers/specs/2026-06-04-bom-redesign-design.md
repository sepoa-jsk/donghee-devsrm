## ② 화면별 반영

### 02-01 품번 목록
- H1110-T100 프로젝트(PJT-520PHEV-FT-2026)의 품번을 표준 BOM 9개(T100, T101, T101A, T101B, T102, T102A, T103, T104, T105)로 교체
- 컬럼 Level/구매구분/목표구매단가/Rev를 표준 BOM 정합
- 다른 프로젝트(NX5, MQ4) 품번 행은 데모용 1~2개씩 유지

### 02-02 품번·BOM·도면 상세 (핵심)
- **좌측 BOM 트리**: 표준 구조로 교체. 루트 H1110-T100 펼침 → T101(→T101A,T101B), T102(→T102A), T103, T104, T105
- **eBOM/mBOM 탭 추가** (트리 상단): eBOM(설계 기준, 기본) / mBOM(제조 기준). mBOM 전환 시 내제 가공품(T101A/B, T102A) 강조 + 공정 순서 표시(간단 라벨)
- **PLM 연계 상태 배지**: 우측 상세 헤더에 연동(초록)/미연동(회색)/충돌(빨강). H1110-T100 = 연동, T105 Hose = 미연동(신규)
- **영향 카드 클릭 링크화**: 재료비(03-07), RFQ(04-04), 설계변경(08-01) → 실제 href
- **키체인 컨텍스트 바** 추가(상단): 프로젝트→품번→Rev→적용시점 + 링크

### 02-03 도면 리비전 이력
- H1110-T100 Rev.A/Rev.B 행 정합(무게 2.70→2.85), 하위 T101~T105 Rev.A 행 추가
- 다른 프로젝트 행은 1~2개 유지

### 02-04 BOM 트리 뷰
- BOM_DATA를 표준 구조로(T104 Pipe, T105 Hose 명칭/무게 반영)
- **eBOM/mBOM 실제 전환**: mBOM 선택 시 내제품 강조 + 공정 컬럼(또는 행 음영) 토글
- **적용시점(effective date) 컬럼 추가**: 각 품번 Rev 적용일 (T100 Rev.B 2026-03-10, 하위 2026-01-15)
- **프로젝트 카드 클릭 링크화**: [01-04 프로젝트 상세] 이동
