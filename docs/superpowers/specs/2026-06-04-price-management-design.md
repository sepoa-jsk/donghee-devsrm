## 06-02 확정단가 등록/상세

**파일:** `mockups/06_price/06-02_price_detail.html`

### 상단 품번 정보 카드 (읽기 전용)
- 품번: H1110-T100 / 품명: Frame Sub Ass'y / 협력사: A사 / 프로젝트: PJT-520PHEV-FT-2026
- 도면 리비전: Rev.C / LME 기준월: 2026-03

### 단가 종류 탭
세 탭 모두 공통으로 상단에 확정단가(원) 입력, 적용기간 입력 존재.

**가단가 탭**
- 임시 적용 이유 선택 (드롭다운: 개발 초기 / 도면 미확정 / 협의 진행 중)
- 정단가 전환 예정일 입력
- 전환 조건 메모 (텍스트)

**정단가 탭** (기본 선택)
- LME 스냅샷: Al 6061 / 2,650 USD/MT / 기준월 2026-03
- 환율 스냅샷: 1,380 KRW/USD
- 프리미엄: 120 USD/MT
- 도면 리비전 스냅샷: Rev.C
- 조정 조건: LME ±10% 초과 시 재협의 (텍스트)

**사급단가 탭**
- 지급소재 분리표: 소재명 / 단위 / 수량 / 단가 / 금액
- 가공비 단가 (원): 별도 입력
- 최종 사급단가 = 지급소재 합계 + 가공비

### 하단
- 결재선 5단계: 담당→팀장→구매부장→CFO→대표이사
- [이력 보기] 링크 → 06-03

## 06-06 임의정산(소급정산) 관리

**파일:** `mockups/06_price/06-06_retroactive.html`

### 검색바
- 좌: 품번 검색 + 정산방식 필터(세금계산서/상계) + 날짜 범위
- 우: [정산 등록] 버튼 → 하단 슬라이드 패널 열기

### 그리드
| 품번 | 소급기간 | 기준단가 | 변경단가 | 차액/개 | 소급수량 | 총 차액 | 정산방식 | 상태 |

하단 합계 행 고정.

### 정산 등록 슬라이드 패널 (우측)
1. 품번 선택 → 단가 이력 자동 조회
2. 소급기간 입력 (날짜 범위)
3. 수량 입력 → 차액 자동 계산 표시
4. 정산방식 선택 (세금계산서 / 상계 처리)
5. [저장] 버튼
