> 단계: Phase 1 HTML 목업

## 1. 제품 개요

### 1.1 한 줄 정의

고객 RFQ부터 LME 기반 재료비 산출, 협력사 견적 비교, 품질·설계·영업 검토, 단가 확정, ERP 양산 전환까지를 **품번 기준**으로 연결하여 개발단계 원가와 구매 의사결정을 통제하는 시스템.

### 1.2 도메인 원칙 (불변)

| # | 원칙 | 설명 |
|---|---|---|
| P1 | 목표가 선행 | 협력사 견적 접수 전 내부 목표 구매단가 확정 필수. 협력사 비공개. |
| P2 | LME 기반 재료비 | `투입중량 × 소재단가 − 스크랩중량 × 회수단가`. 소재단가 = `LME × 환율 + 프리미엄 + 부대비`. |
| P3 | 품번 기준 추적 | `프로젝트 → 품번 → 도면리비전 → 적용시점` 키 체인. |
| P4 | Breakdown 검증 | 협력사 견적은 LME·환율·프리미엄·투입중량·스크랩·가공비·물류비·관리비/이윤 항목별 수령. |
| P5 | 다부서 검토 | 업체 선정은 설계·품질·영업 검토 결재 필수. |
| P6 | 변경 추적 | 도면 리비전·LME·환율 변동 시 재료비·단가 자동 재계산 + 이력 보존. |
| P7 | ERP 전환 통제 | 체크리스트 전 항목 충족 전 ERP 전송 버튼 비활성. |

### 3.4 03. 원가관리

#### 03-01 목표가 설정 목록
- 검색: 프로젝트, 품번, 차수, 상태
- 그리드: 차수 / 기준일 / 고객목표가 / 마진율 / 목표구매단가 / 상태

#### 03-02 목표가 설정 상세
- 입력: 고객목표판매가·목표마진율·내부비용
- 자동계산: 허용총원가·목표구매단가
- **단계별 목표원가 차수 관리 연결**: 03-09 화면과 연동, 차수별 이력 조회 가능
- 산정 근거 텍스트, 승인 워크플로우

#### 03-03 LME 기준정보 목록
- 소재·기준월별 목록, 적용 현황

#### 03-04 LME 기준정보 등록
- 입력: 소재·LME기준가·기준월·환율·프리미엄·부대비
- 자동계산: 소재단가 (KRW/kg)
- 적용기간 입력, 충돌 검증

#### 03-05 환율 관리
- 월별 환율 등록, 적용기간, 전망치 입력

#### 03-06 재료비 산출 목록
- 프로젝트·품번별 재료비 산출 현황

#### 03-07 재료비 산출 상세
- 좌: 도면/소재 정보 (읽기)
- 중: 사용량 입력 (투입중량·스크랩·회수단가)
- **공법 선택**: PRESS / 절삭가공 / 냉간단조 / 열간단조 / PLASTIC INJECTION / 다이캐스팅 / 기타
  - 공법에 따라 가공비 산출 공식 자동 전환
- 우: 계산 결과 카드 (재료비 11,168원)
- 하단: 결재선

#### 03-08 목표가 대비 원가 분석
- KPI 3장: 목표 / 예상 / 차이
- 누적 막대: 재료비/가공비/물류/관리
- 부서별 검토 의견 영역 (설계·구매·영업·개발)

#### 03-09 단계별 목표원가 차수 관리 _(신규)_
- 개발 단계(RFQ→SOP)별 목표원가 차수 이력 관리
- 차수별 목표구매단가 변경 이력, 변경 사유, 승인자
- 차수 비교 테이블 (차수1 vs 차수2 등)
- 03-02 목표가 설정 상세와 연동

#### 03-10 원가절감 목표 부서별 배부 _(신규)_
- 총 원가절감 목표액 입력
- 부서별(설계/구매/품질/영업) 배부 비율 설정
- 배부 목표 vs 실적 비교 테이블
- 절감 항목별 상세 등록

#### 03-11 현상원가 차이 분석 _(신규)_
- 견적 접수 시점의 현상원가 vs 목표원가 차이 분석
- 항목별 차이 원인 분류 (재료비 / 가공비 / 물류비 / 관리비 / 기타)
- 차이 원인별 조치 계획 입력
- 개선 후 예상원가 재산출

### 3.5 04. 개발구매

#### 04-01 구매대상 품목 확정
- BOM 품목 중 구매대상 선택 확정
- 그리드: 품번 / 품명 / 재질 / 도면Rev / 투입중량 / 재료비 / 목표구매단가 / 구매구분 / 확정상태
- 확정 후 RFQ 생성 버튼 활성

#### 04-02 RFQ 목록
- 검색: 프로젝트, 품번, 발송일, 마감일, 상태
- 그리드: RFQ번호 / 품번 / 발송일 / 마감일 / 대상 협력사 수 / 제출 수 / 상태

#### 04-03 RFQ 생성/발송
- 품번 정보 카드
- 대상 협력사 다중 선택 그리드
- 마감일, LME 기준 자동 포함 안내
- **4년치 연도별 단가 + 인하율 + 옵션율 입력 양식**: 1차년~4차년 단가 및 연도별 인하율(%), 옵션 조건율(%) 입력란
- 필수 입력항목 체크박스 리스트

#### 04-04 RFQ 상세 (대상 협력사 진행상태)
- 협력사별 견적 제출 상태 (미제출/제출/확인 중/완료)
- 마감일 D-day 표시

#### 04-05 협력사 견적 제출 (협력사 포털)
- 협력사용 간소화 입력 폼
- LME/환율 기준 확인, Breakdown 항목별 입력
- 목표가 비공개

#### 04-06 견적 Breakdown 비교
- 상단: 품번/RFQ 정보
- 메인 테이블: 항목 × (내부기준 / A사 / B사 / C사)
- 차이 셀 컬러 음영 (높음=주황, 낮음=회색)
- **4년간 총 구매금액 비교 컬럼 추가**: 연도별 단가 × 연간수량 합산 기준 협력사별 총액 비교
- 하단: 종합 판단 (가격/품질/납기/금형비)

#### 04-07 부서별 검토 (설계/품질/영업 탭)
- 탭: 설계 / 품질 / 영업 손익
- 영업 손익: 고객목표가·구매단가·내부비용·예상마진 자동계산
- **심의입찰 평가표 가중치 항목 추가**: 별도 탭으로 심의입찰 시 가중치 평가 항목 포함 (가격 40% / 품질 30% / 납기 15% / 기술력 15% 기본값, 수정 가능)

#### 04-08 협상 이력 관리
- 라운드별 행 추가
- 항목별 최초→협상후 비교, 절감액 표시
- 협상 근거 텍스트, 첨부

#### 04-09 원가절감안 등록
- 절감 항목 등록 (VE/VA/공법변경/소재변경 등)
- 예상 절감액, 적용 시점, 담당 부서

#### 04-10 업체 선정 품의/결재
- 결재문서 자동 생성 영역
- **선정 유형 필드**: 전략적 선정 / 경쟁 입찰 / 심의입찰 중 선택
- **이원화 여부 필드**: 단일 업체 / 이원화 / 다원화 선택, 이원화 시 업체별 비율 입력
- 수평 결재선 7단계
- 반려 사유 입력 모달

#### 04-11 심의입찰 평가 (가중치 기반) _(신규)_
- 평가 항목 구성: 가격(40%) / 품질(30%) / 납기(15%) / 기술력(15%)
- 항목별 배점 가중치 수정 가능 (합계 100% 자동 검증)
- 협력사별 평가 점수 입력 → 가중 합산 점수 자동 계산
- 평가 결과 순위 테이블, 최종 선정 의견 입력

#### 04-12 업체변경 / 이원화·다원화 품의 _(신규)_
- 기존 선정 업체 정보 조회
- 변경·추가 사유 입력 (품질 이슈 / 원가 절감 / 공급 안정성 / 기타)
- 이원화·다원화 시 업체별 공급 비율 설정
- 결재선 연동

### 3.6 05. 품질관리

#### 05-01 협력사 품질 평가
- 협력사별 Q 지수 (5스타 평가 기반)
- 평가 항목: 납기 준수율 / 불량률 / 시스템 운영 / 개선 활동

#### 05-02 APQP 일정 관리
- APQP 5단계 체크리스트 (계획→제품설계→공정설계→유효화→양산)
- 단계별 완료 여부, 지연 표시

#### 05-03 샘플·금형 일정
- 초도샘플·초도금형·양산금형 일정 관리
- 진행 상태, 지연 건수 표시

#### 05-04 PPAP 문서 관리
- **PSO 12단계 문서 워크플로우**: 설계기록 / DFMEA / PFMEA / 공정흐름도 / Control Plan / 측정시스템 분석 / 치수결과 / 재질성적서 / 초도량 생산시험 / 외관승인 / 샘플부품 / 포장규격 — 12종 문서 상태 관리
- 협력사 업로드 / 품질 검토 / 승인 워크플로우 (제출→검토→반려/승인)

#### 05-05 EO 접수 현황 (초도/설변) _(신규)_
- EO(Engineering Order) 접수 목록
- 유형 구분: 초도 EO / 설계변경 EO
- EO번호 / 발행일 / 유효부품 / 적용 시점 / 처리 상태
- 설계 변경 연동 (08-01 화면 트리거)

#### 05-06 개발 요청서 작성 (가단가 결정) _(신규)_
- 개발 요청서 양식 (DR: Development Request)
- 가단가(임시 단가) 결정 근거 입력
- 적용 기간, 조건 입력
- 06-04 가단가 품의와 연동

#### 05-07 설계 변경 통보 _(신규)_
- 설계 변경 내용 공식 통보 양식 (협력사·내부 발송)
- 변경 전/후 비교 (도면번호·리비전·변경 내용)
- 통보 대상 협력사 선택, 발송 이력 기록

### 3.7 06. 단가관리

#### 06-01 확정단가 목록
- 품번·프로젝트·적용기간별 단가 목록
- 단가 종류별 필터 (가단가/정단가/사급단가)

#### 06-02 확정단가 등록/상세
- 단가 + 기준 스냅샷 (LME·환율·프리미엄·도면리비전)
- **단가 종류 구분 필드**: 가단가(임시) / 정단가(확정) / 사급단가(지급 소재 포함) 선택
- 적용 기간, 조정 조건

#### 06-03 단가 이력 (시계열)
- 품번별 단가 변경 이력 (시계열 테이블)
- 변경 사유, 기준 스냅샷 연결

#### 06-04 가단가 / 정단가 / 사급단가 품의 _(신규)_
- 단가 유형별 품의 양식
- 가단가: 임시 적용 기간 + 정단가 전환 조건
- 정단가: 기준 LME·환율·원가 스냅샷 확정
- 사급단가: 지급 소재 항목 분리, 가공비만 단가 처리
- 결재선 연동

#### 06-05 단가 미결 현황 모니터링 _(신규)_
- 정단가 미확정 품번 목록
- 가단가 적용 기간 초과 경고
- 미결 기간별 현황 (30일 이내 / 30~60일 / 60일 초과)
- 조치 담당자, 예상 확정일 표시

#### 06-06 임의정산(소급정산) 관리 _(신규)_
- 단가 소급 적용 대상 품번 조회
- 소급 기간, 수량, 차액 자동 계산
- 정산 방식: 세금계산서 발행 / 상계 처리
- 정산 완료 이력 보존

### 3.9 08. 변경관리

#### 08-01 설계 변경 영향 분석
- 변경 전/후 비교 (리비전·중량·재료비·예상단가)
- 영향 협력사·확정단가 리스트
- 재협의 트리거 버튼

#### 08-02 LME 변동 영향 분석
- LME 입력 (시뮬레이션 또는 실제 변경)
- **연동대상 일괄 추출**: 적용 소재를 사용 중인 품번 자동 추출
- **변동금액 자동 계산**: 품번별 `(변경 LME − 기준 LME) × 투입중량 × 환율` 자동 산출
- **ERP 일괄 반영 3단계**: ① 영향 품번 확인 → ② 단가 변경 품의 생성 → ③ ERP 전송
- 단가 조정 조건 자동 매칭

#### 08-03 환율 변동 영향 분석
- 환율 입력 (시뮬레이션 또는 실제 변경)
- 영향 품번 그리드: 품번 / 기존 재료비 / 변경 재료비 / 증감액

#### 08-04 단가 변경 이력
- 품번별 단가 변경 전 이력 보존

### 3.10 09. 협력사 관리 (Supplier) — 라이프사이클 6단계 구조

#### 09-01 협력사 등록 신청 (목록) — ①발굴·등록
- 신규 협력사 등록 신청 목록 (상태별 필터: 신청 / 심사중 / 승인 / 반려)
- [협력사 등록] 버튼 → 09-02 신청 폼 이동
- 그리드: 업체코드 / 업체명 / 품목군 / 신청일 / 사전평가 점수 / 등록 상태 / 담당자

#### 09-02 협력사 신규 등록 (신청 폼) — ①발굴·등록
- 업체 기본 정보 입력 (업체명·사업자번호·대표자·주소·품목군 등)
- 좌측 탭 6개: 일반정보 / 담당자정보 / 경영정보 / 영업현황 / 인증정보 / 신용평가정보
- 제출 시 09-01 목록에 "신청" 상태로 등록

#### 09-03 협력사 적격성 심사·승인 — ②적격성 심사
- 09-01에서 업체 클릭 시 진입 (URL 파라미터: code·name·st·mgr·apply·score)
- 일반정보~신용평가정보 탭 유지 + **심사결과·결재** 탭 추가
- 심사 체크리스트: 재무건전성·기술역량·품질시스템·생산능력·협력가능성 각 20점
- 종합 판정: 적격 / 조건부 적격 / 부적격
- 승인 결재선 4단계 (담당→팀장→부장→임원)

#### 09-04 정식 협력사 마스터 — ③정식등록
- 적격성 심사 승인 협력사의 정식 등록 현황 목록
- 상단 요약 KPI: 총 협력사 / 전략협력사 / 일반협력사 / ERP 미연계
- 거래구분(전략/일반) 세그먼트 칩 필터
- 그리드: 협력사코드 / 업체명 / 사업자번호 / 품목군 / 거래구분 / 현재등급 / ERP연계상태 / 등록일

#### 09-05 협력사 등급평가 (현황) — ⑤성과평가
- 조회 전용 화면 (입력·저장 버튼 없음)
- 상단 요약: 총 218사 / S 1·0% / A 9·4% / B 5·2% / C 3·1% / D 200·92%
- 2단 그룹 헤더 그리드 (원가50/품질30/기술10/개발10/가감점):
  - 원가 세부: 입찰견적(45)·절감실적(3)·개선제안(2)
  - 품질 세부: 협력사평가(10)·공정불량율(20)
  - 기술 세부: 기술능력(5)·제안(5)
  - 개발 세부: 개발능력(5)·업무대응력(5)
  - 가감점: 상생협력(+3)·제재(-10)·납입(-2)·신용평가(-2)
- 필터 칩 "연도 2025 ×" + 등급 필터, 자동 조회

#### 09-06 협력사 사후 평가 (Q-5스타) — ⑥재평가·육성
- Q-5스타 평가 시스템 (품질Q / 납기D / 가격P / 기술T / 시스템S, 각 1~5점)
- 분기별 평가 입력 그리드 (협력사별 × 분기별)
- 등급 자동 산정: S(4.5+) / A(3.5+) / B(2.5+) / C(1.5+) / D(미만)
- 등급별 조치 사항 (우선 발주 / 경고 / 개선 계획 / 거래 중단)
