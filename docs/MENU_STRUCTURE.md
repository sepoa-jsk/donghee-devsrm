## 화면별 핵심 컬럼/구성 (목업 작업 가이드)

### 01-01 고객 RFQ 목록
- 검색: 고객사, 차종, 등록일, 상태
- 그리드: RFQ번호 / 고객사 / 차종 / 품목 / 고객목표가 / 연간수량 / SOP / 등록일 / 상태
- 상태: RFQ등록 / 프로젝트전환대기 / 검토요청 / 프로젝트생성완료

### 01-02 고객 RFQ 등록/상세
- 폼: 고객사·차종·고객RFQ번호·프로젝트명·요청품목·고객목표판매가·연간수량·SOP·첨부
- 액션: 임시저장 / 등록 / 프로젝트 생성

### 01-04 개발 프로젝트 상세 (대시보드)
- KPI 6장: 일정 / 원가 / 구매 / 품질 / 설계 / 영업
- 마일스톤 간트 (도면등록→SOP)
- 담당자 카드, 최근 이슈 리스트

### 02-02 품번·BOM·도면 상세
- 좌측 BOM 트리, 우측 선택 품번 상세
- **eBOM / mBOM 구분 탭**: eBOM(설계 기준) ↔ mBOM(제조 기준) 전환
- **PLM 연계 상태**: 연동/미연동/충돌 표시
- 상세: 품번·품명·도면번호·리비전·재질·LME연동소재·중량(순/투입/스크랩)·구매대상
- 도면 첨부, 리비전 이력 탭

### 02-05 BOM 구성 작성 [신규]
- 최상위 품번(H1110-T100) 아래 자식 품번을 추가/삭제하며 eBOM 트리를 직접 편집·구성
- 툴바: 자식 추가 / 형제 추가 / 들여쓰기 / 내어쓰기 / 위·아래 이동 / 행 삭제
- 행별 인라인 입력: 품번·품명·수량·재질·도면Rev·구매구분(구매대상/내제/사급)·순중량·투입중량 (스크랩 자동)
- 우측 구성 요약: 총 품번 / 최대 레벨 / 총 투입중량 / LME 연동 품번 / 구매구분 분포 / 구성 검증
- 저장 시 02-04 BOM 트리 뷰·03-07 재료비 산출과 연결

### 03-02 목표가 설정 상세
- 입력: 고객목표판매가·목표마진율·내부비용
- 자동계산: 허용총원가·목표구매단가
- **03-09 단계별 목표원가 차수 이력 연동 링크**
- 산정 근거 텍스트, 승인 워크플로우

### 03-04 LME 기준정보 등록
- 입력: 소재·LME기준가·기준월·환율·프리미엄·부대비
- 자동계산: 소재단가 (KRW/kg)
- 적용기간 입력, 충돌 검증

### 03-07 재료비 산출 상세
- 좌: 도면/소재 정보 (읽기)
- 중: 사용량 입력 (투입중량·스크랩·회수단가)
- **공법 선택**: PRESS / 절삭가공 / 냉간단조 / 열간단조 / PLASTIC INJECTION / 다이캐스팅 / 기타
- 우: 계산 결과 카드 (재료비 11,168원)
- 하단: 결재선

### 03-08 목표가 대비 원가 분석
- KPI 3장: 목표 / 예상 / 차이
- 누적 막대: 재료비/가공비/물류/관리
- 부서별 검토 의견 영역 (설계·구매·영업·개발)

### 03-09 단계별 목표원가 차수 관리 [신규]
- 검색: 프로젝트, 품번, 차수범위
- 그리드: 차수 / 개발 단계 / 기준일 / 목표구매단가 / 변경 사유 / 승인자
- 차수 비교 테이블 (복수 차수 선택 후 항목별 비교)
- 03-02 목표가 설정 상세에서 진입 가능

### 03-10 원가절감 목표 부서별 배부 [신규]
- 총 절감 목표액 입력 (프로젝트·품번 연결)
- 부서별 배부 비율 슬라이더 (합계 100% 자동 검증)
- 그리드: 부서 / 배부 목표액 / 실적액 / 달성률 / 주요 절감 항목

### 03-11 현상원가 차이 분석 [신규]
- 상단 KPI: 현상원가 / 목표원가 / 차이금액
- 항목별 차이 테이블: 항목 / 현상 / 목표 / 차이 / 차이 원인
- 차이 원인별 조치 계획 입력
- 조치 후 예상원가 재산출

### 04-03 RFQ 생성/발송
- 품번 정보 카드, 대상 협력사 다중 선택
- **4년치 연도별 단가 입력표**: 1차년~4차년 × 단가/인하율(%)/옵션율(%)
- LME 기준 자동 포함 안내, 필수 구비서류 체크리스트

### 04-06 견적 Breakdown 비교 ★ 데모 핵심
- 상단: 품번/RFQ 정보
- 메인 테이블: 항목 × (내부기준 / A사 / B사 / C사)
- **4년 총구매금액 비교 컬럼**: 연도별 단가 × 연간수량 합산
- 차이 셀 컬러 음영, 하단 종합 판단

### 04-07 부서별 검토 (탭)
- 탭: 설계 / 품질 / 영업 손익 / **심의입찰 평가**
- 심의입찰 평가 탭: 가중치 항목(가격40/품질30/납기15/기술15) 입력

### 04-08 협상 이력 관리
- 라운드별 행 추가
- 항목별 최초→협상후 비교, 절감액 표시
- 협상 근거 텍스트, 첨부

### 04-10 업체 선정 품의/결재
- 결재문서 자동 생성
- **선정 유형**: 전략적 선정 / 경쟁 입찰 / 심의입찰
- **이원화 여부**: 단일 / 이원화 / 다원화 + 비율 입력
- 수평 결재선 7단계, 반려 사유 모달

### 04-11 심의입찰 평가 (가중치 기반) [신규]
- 평가 항목 + 가중치 편집 테이블 (합계 100% 자동 검증)
- 협력사별 평가 점수 입력 그리드
- 가중 합산 점수 자동 계산, 순위 표시
- 최종 선정 의견 입력

### 04-12 업체변경 / 이원화·다원화 품의 [신규]
- 기존 선정 업체 조회
- 변경·추가 사유 선택 (품질 이슈 / 원가 절감 / 공급 안정성 / 기타)
- 이원화 시 업체별 공급 비율 입력
- 결재선 연동

### 05-04 PPAP 문서 관리
- **PSO 12단계 워크플로우**: 설계기록·DFMEA·PFMEA·공정흐름도·Control Plan·측정시스템·치수결과·재질성적서·초도량 시험·외관승인·샘플부품·포장규격
- 협력사 업로드 / 품질 검토 / 반려·승인 3단계

### 05-05 EO 접수 현황 (초도/설변) [신규]
- 검색: EO번호, 유형, 발행일, 처리 상태
- 그리드: EO번호 / 유형(초도/설변) / 발행일 / 유효 품번 / 적용 시점 / 처리 상태
- 설계 변경 연동 (08-01 트리거)

### 05-06 개발 요청서 작성 (가단가 결정) [신규]
- DR(Development Request) 양식
- 가단가 결정 근거 (재료비·가공비·예상 정단가 기준 범위)
- 적용 기간, 조건 입력
- 06-04 가단가 품의 연동

### 05-07 설계 변경 통보 [신규]
- 변경 전/후 비교 (도면번호·리비전·변경 내용)
- 통보 대상 협력사 선택 (체크박스 다중)
- 발송 이력 기록

### 06-02 확정단가 등록/상세
- 단가 + 기준 스냅샷 (LME·환율·프리미엄·도면리비전)
- **단가 종류**: 가단가(임시) / 정단가(확정) / 사급단가(지급소재 포함)
- 적용 기간, 조정 조건

### 06-04 가단가/정단가/사급단가 품의 [신규]
- 단가 유형 선택 탭
- 가단가: 임시 적용 기간 + 정단가 전환 조건
- 정단가: LME·환율·원가 스냅샷 확정
- 사급단가: 지급 소재 항목 분리, 가공비만 단가 처리
- 결재선 연동

### 06-05 단가 미결 현황 모니터링 [신규]
- 검색: 미결 기간, 담당자, 프로젝트
- 그리드: 품번 / 적용 중 단가 종류 / 가단가 적용일 / 미결 일수 / 예상 확정일 / 조치 담당자
- 미결 기간 30/60일 초과 시 컬러 경고

### 06-06 임의정산(소급정산) 관리 [신규]
- 소급 대상 품번 조회 (단가 변경 이력 연동)
- 소급 기간·수량 입력, 차액 자동 계산
- 정산 방식: 세금계산서 발행 / 상계 처리
- 정산 완료 이력

### 07-01 ERP 양산 전환 체크리스트
- 10개 체크 항목 (목표가/재료비/품질/단가/결재 등)
- 모두 충족 시 [ERP 전송] 버튼 활성
- 전송 데이터 미리보기

### 08-01 설계 변경 영향 분석
- 변경 전/후 비교 (리비전·중량·재료비·예상단가)
- 영향 협력사·확정단가 리스트
- 재협의 트리거 버튼

### 08-02 LME 변동 영향 분석
- LME 입력 (시뮬레이션 또는 실제 변경)
- **1단계: 연동대상 일괄 추출** — 해당 소재 사용 품번 자동 조회
- **2단계: 변동금액 자동 계산** — 품번별 `(변경 LME − 기준 LME) × 투입중량 × 환율`
- **3단계: ERP 일괄 반영** — 단가 변경 품의 생성 → ERP 전송
- 단가 조정 조건 자동 매칭

### 09-01 협력사 사전 등록·평가 [신규]
- 검색: 업체명, 품목군, 등록 상태
- 그리드: 업체코드 / 업체명 / 품목군 / 신청일 / 평가 점수 / 등록 상태
- 상세: 기본 정보 + 사전 평가 체크리스트 (재무·기술·품질·생산)
- 등록 상태: 신청 / 심사중 / 승인 / 반려

### 09-02 협력사 사후 평가 (Q-5스타) [신규]
- 평가 항목: 품질(Q) / 납기(D) / 가격(P) / 기술(T) / 시스템(S) — 각 1~5점
- 분기별 평가 입력 그리드 (협력사별 × 분기별)
- 등급 자동 산정: S(4.5+) / A(3.5+) / B(2.5+) / C(1.5+) / D(미만)
- 등급별 조치 사항 연결 (우선 발주 / 경고 / 개선 계획 / 거래 중단)
